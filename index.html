Fuel UX extends Twitter Bootstrap with additional lightweight JavaScript controls.

Download Fuel UX

Enhance your Bootstrap project.

Fuel UX includes easy installation into web projects, integrated scripts for customizing Bootstrap and Fuel UX, simple updates, and solid optimization for deployment.

Preprocessors

Like it's parent library, Fuel UX includes support for vanilla CSS and the two most popular CSS preprocessors, Less and Sass.

UMD Compliant

Fuel UX utilizes the UMD standard, providing compatibility with AMD or CommonJS if desired.

Docs & Unit Tests

Thorough documentation is provided, and all controls are unit-tested to ensure quality.

Fuel UX is open source. It's hosted, developed, and maintained on GitHub.
