    Usage

    A pillbox allows user tag management enabling features such as typahead and tag editing.

    Trigger the Pillbox via JavaScript:

    {% highlight js %}$('#MyPillbox').pillbox();{% endhighlight %}

    Markup

    Add class="pillbox" to an a div within a class="fuelux" container.

      Methods

      .spinbox('addItems')

      Adds Items to the pillbox programatically. The first parameter is a 1 based index that represents the position the items will be placed in the pill order. If index is not specified items will be added after last existing item.

      {% highlight js %}$('#element').spinbox('addItems', index, [{ text: '', value: '' }, { ... }]) {% endhighlight %}

      Alternatively items can be added as individual parameters.

      {% highlight js %}$('#element').spinbox('addItems', { text: '', value: '' },{ ... },{ ... }...) {% endhighlight %}

      .spinbox('removeItems')

      Adds Items to the pillbox programatically. The first parameter is a 1 based index that represents the location of the first element to be removed. The second parameter is the number of items that will be removed.

      {% highlight js %}$('#element').spinbox('removeItems', index, count) {% endhighlight %}

      .spinbox('items')

      Returns an array of objects, one per item, each containing the jQuery data() contents of the item which includes data-* attributes plus a text property with the label's visible text.

      {% highlight js %}$('#element').spinbox('items') {% endhighlight %}

      .spinbox('itemCount')

      Returns an int representing the current number of items.

      {% highlight js %}$('#element').spinbox('itemCount') {% endhighlight %}

      .spinbox('removeBySelector')

      Removes an item based on selector matching that of the provided selector string parameter.

      {% highlight js %}$('#element').spinbox('removeBySelector') {% endhighlight %}

      .spinbox('removeByText')

      Removes an item with contained text matching that of the provided text string parameter.

      {% highlight js %}$('#element').spinbox('removeByText') {% endhighlight %}

      .spinbox('removeByValue')

      Removes an item with value matching that of the provided value string parameter.

      {% highlight js %}$('#element').spinbox('removeByValue') {% endhighlight %}

      Options

      Options can be passed via data attributes or JavaScript. For data attributes, append the option name to data-, as in data-selected="".

      Name type default description
      acceptKeyCodes array [13, 188] Key codes for keys that trigger an add pill event. Default keys 13 (for enter) and 188 (for camma).
      edit boolian false Enables user edited pills.
      onAdd function undefined Function that runs whenever a pill is added. {% highlight js %}function(data,callback){}{% endhighlight %} The data parameter contains an array of the elements being added. The callback parameter is a function that provides asynchronous support for the add functionality. In order for items to be added, the callback must be run and provided an array of items to be added. If you would not like to modify the list of items to be added, simply proved data as the parameter for the callback function. For example: callback(data)
      onRemove function undefined Function that runs whenever a pill is removed. {% highlight js %}function(data,callback){}{% endhighlight %} The data parameter contains an array of the elements being removed. The callback parameter is a function that provides asynchronous support for the remove functionality. In order for items to be removed, the callback must be run and provided an array of items to be removed. If you would not like to modify the list of items to be removed, simply proved data as the parameter for the callback function. For example: callback(data)
      onKeyDown function undefined Function that runs whenever a keydown event is triggered. {% highlight js %}function(event,data,callback){}{% endhighlight %} The event parameter contains the event object. The data parameter contains an array of the elements being removed. The callback parameter is a function that provides asynchronous support for the typahead functionality. The callback function must be run in order for the typahead dropdown with values. Provided the callback with an array of items to display in the typahead dropdown. For example: callback({data:[{text: '',value:''},{text: '',value:''}]})

      Events

      Event Type Description
      clicked This event is triggered when a pill is clicked. An object containing information on the clicked pill is returned.
      added this event is triggered whenever a pill is added. An object containing information on the added pill is returned.
      removed this event is triggered whenever a pill is removed. An object containing information on the removed pill is returned.
      edited this event is triggered whenever a pill is edited. An object containing information on the edited pill is returned.
