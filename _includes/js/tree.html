    Usage

    The tree is a categorical selection interface that allows for interaction and selection of nested elements.

    Via JavaScript

    Call the tree control via JavaScript:

    {% highlight js %}$('#MyTree').tree({ dataSource: dataSource });{% endhighlight %}

    Markup

    Add class="tree" to an a div within a class="fuelux" container. Markup varies slightly depending on the type of tree you are using.

      Methods

      .tree('destroy')

      Removes all functionality, jQuery data, and the markup from the DOM. Returns string of control markup.

      {% highlight js %}$('#element').tree('destroy');{% endhighlight %}

      .tree('selectedItems')

      Returns an array containing selected items and associated data.

      {% highlight js %}$('#element').tree('selectedItems'){% endhighlight %}

      Options

      Options can be passed via data attributes or JavaScript. For data attributes, append the option name to data-, as in data-selected="".

      Name type default description
      multiSelect boolean false Allows multiple tree items to be selected at once.
      cacheItems boolean true When a folder is closed and re-opened sub-items will not be refreshed.
      folderSelect boolean false Enables folder selection.

      Events

      Event Type Description
      loaded.fu.tree This event is fired when sub-content has been is loaded.
      selected.fu.tree This event is fired when item/items has been selected. An object containing {info: data} is returned. data represents an array of selected items
      opened.fu.tree This event is fired when a sub-folder is opened. An object containing folder information is returned.
      closed.fu.tree This event is fired when a sub-folder is closed. An object containing folder information is returned.
