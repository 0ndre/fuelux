Datepickers [STUB] datepicker.js

Examples

Choose a data from a calendar dropdown or use moment.js to format and parse various date formats.

To support various date formats around the world, datepicker has an optional dependency on moment.js. Download moment.js (with languages).

Usage

Via data-attributes

The datepicker will automatically instantiate any datepickers with data-initialize="datepicker". If the control markup is added after page load with data-initialize="datepicker", this control will be initialized on mousedown.

Via JavaScript

Call the datepicker control via JavaScript:

{% highlight js %}$('#MyDatepicker').datepicker();{% endhighlight %}

Markup

Add class="datepicker" to an input and a dropdown within a class="fuelux" container.

Options

Options can be passed via data attributes or JavaScript. For data attributes, append the option name to data-, as in data-selected="".

Name type default description
date date new Date() If date is set to null when the datepicker is initialized the getDate method will return null until a date is selected (programmatically or by the user). Otherwise, the date will be parsed.
createInput object false Allows control to generate HTML Markup. This can be true, false, or an object like below: { native: false, dropDownBtn: false, inputSize: 'span3' } native can have a value of true or false. If true, dropDownBtn will be ignored dropDownBtn can have a value of true or false inputSize can either be a CSS class or a integer.
dropdownWidth integer 170 Sets pixel width of dropdown.
restrictDateSelection boolean true Disables ability to select dates in the past. Based on today's date.
parseDate date return new Date( date ); Function allows custom date parsing. A date is passed as a parameter. A Date Object should be returned by this function. Do not overwrite this function if using Moment.js.
formatDate string MM-DD-YYYY Function allows custom formatting of date. Will never modify the main Date Object. A date is passed as a parameter. A formatted date should be returned by this function. Do not overwrite this function if using Moment.js.
blackoutDates boolean true Function restricts date selection. A date is passed as a parameter. Boolean value should be returned false
momentConfig Object true Handles culture and format if Moment.js is available { culture: 'en', formatCode: 'L' }

Methods

$().datepicker(options)

Attaches a datepicker to an input and a dropdown menu.

.datepicker('destroy')

Removes all functionality, jQuery data, and the markup from the DOM. Returns string of control markup.

{% highlight js %}$('#element').datepicker('destroy');{% endhighlight %}

.datepicker('disable')

Disables the component

{% highlight js %}$('#element').datepicker('disable'){% endhighlight %}

.datepicker('enable')

Enables the component

{% highlight js %}$('#element').datepicker('enable'){% endhighlight %}

.datepicker('getDate')

Returns the selected date (not formatted)

{% highlight js %}$('#element').datepicker('getDate'){% endhighlight %}

.datepicker('getFormattedDate')

Returns the selected formated date

{% highlight js %}$('#element').datepicker('getFormattedDate'){% endhighlight %}

.datepicker('setDate')

Set's datepicker to date provided. Date provided must be a valid Date Object

{% highlight js %}$('#element').datepicker('setDate', new Date() ){% endhighlight %}

.datepicker('getCulture')

Returns the culture the datepicker is currently using. Only available using Moment.js with Langs

{% highlight js %}$('#element').datepicker('getCulture'){% endhighlight %}

.datepicker('setCulture')

Updates the culture the datepicker uses. Only available using Moment.js with Langs

{% highlight js %}$('#element').datepicker('setCulture'){% endhighlight %}

.datepicker('getFormatCode')

Returns the format code the datepicker is currently using. Only available using Moment.js with Langs

{% highlight js %}$('#element').datepicker('getFormatCode'){% endhighlight %}

.datepicker('setFormatCode')

Updates the format code the datepicker uses. Only available using Moment.js with Langs

{% highlight js %}$('#element').datepicker('setFormatCode'){% endhighlight %}

Events

Event Type Description
changed.fu.datepicker This event is fired when the date value has changed.
inputParsingFailed.fu.datepicker This event is fired when an invalid date is parsed on blur of the datepicker. Only available using Moment JS with Langs
