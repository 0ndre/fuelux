    Options

    Options can be passed via Javascript upon initialization.

    Name type default description
    list_columnRendered function null If set, this function will be called after the repeater renders each column. Passes a helpers object and callback function as arguments. The helpers object has the parent container and current column item as attributes, if available. Once ready, the callback function must be called in order to continue with rendering.
    list_columnSizing boolean true Dictates whether the repeater should run the intelligent column resizing algorithm. Accounts for columns with set widths, only sizing those without. Setting this to false will disable this feature entirely.
    list_columnSyncing boolean true Dictates whether the repeater should run the post-render column syncing algorithm. This modifies column header widths to match column widths that have expanded due to content. Setting this to false will disable this feature entirely.
    list_infiniteScroll boolean or object false Dictates whether the repeater list view should utilize infinite scrolling instead of pagination. If set to true, infinite scrolling will be enabled with default settings. Additionally, it can be set to an object with attributes matching the options available in the infinite scroll control for further customization. (dataSource option will be ignored)
    list_noItemsHTML string or jQuery object '' Specifies what gets displayed if there are no items returned from the dataSource. Can be a string or jQuery object.
    list_selectable boolean or string false Specifies whether the rendered items rows can be selectable. If set to true, only one row can be selected at a time. If set to 'multi', multiple rows can be selected at once.
    list_sortClearing boolean false Specifies whether sortable columns can be "cleared" by a third click (ex: one click, sort ASC. second click, sort DESC. third click, no sorting)
    list_rowRendered function null If set, this function will be called after the repeater renders each row. Passes a helpers object and callback function as arguments. The helpers object has the parent container and current row item as attributes, if available. Once ready, the callback function must be called in order to continue with rendering.
