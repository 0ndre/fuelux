Alignment

By default, a dropdown menu is automatically positioned 100% from the top and along the left side of its parent. Add .dropdown-menu-right to a .dropdown-menu to right align the dropdown menu.

Usage

Via JavaScript, the checkbox plugin toggles hidden content (dropdown menus) by toggling the .open class on the parent list item. When opened, the plugin also adds .dropdown-backdrop as a click area for closing dropdown menus when clicking outside the menu. Note: The data-toggle=dropdown attribute is relied on for closing dropdown menus at an application level, so it's a good idea to always use it.

Options

Options can be passed via data attributes or JavaScript. For data attributes, append the option name to data-, as in data-selected-item="".

Name type default description
selected-item boolean false Declare value to initialize with

Methods

$().combobox('disable')

Disables the component

$().combobox('enable')

Disables the component

$().combobox('selectedItem')

Returns a key value object
